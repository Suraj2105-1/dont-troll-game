# Don't Troll Game

A fun and challenging web-based game featuring tricky traps, multiple levels, and custom achievements!

## Features
- **Challenging Gameplay:** Navigate through levels filled with unpredictable traps.
- **Level Builder:** Create and play your own custom levels.
- **Achievements:** Unlock various achievements as you progress.
- **Leaderboards:** Compete for the top score.
- **Cross-Platform:** Playable on web and bundled for Android via Capacitor.

## How to Play
Open `index.html` in your web browser to start the game.

## Technologies Used
- HTML5 / CSS3
- Vanilla JavaScript
- Capacitor (for Android support)
